Demo de page MVC en Spring Boot

  • Pour les curieux on a un exemple de MVC avec Spring Boot
  • Le contrôleur ControllerDemoMVC contient la méthode index qui répond en GET sur la racine du site (/)
  • Si le type de retour est String sans annotation, cela indique à Spring Boot de chercher un gabarit html
  • Si on retourne index Spring Boot va chercher un gabarit index.html dans les ressources statiques
  • le fichier index.html est un gabarit ThymeLeaf
