Incluant la documentation de l'API

API pour les comptes

Créer un compte :: POST /id/inscription


{
  "nom" : "MrPipo",
  "motDePasse" : "pipo",
  "confirmationMotDePasse" : "pipo"
}

Se connecter avec un compte existant :: POST /id/connexion


{
  "nom" : "MrPipo",
  "motDePasse" : "pipo"
}

Se déconnecter :: POST /id/deconnexion

API pour les tâches

Ajouter une tâche au compte actuel (nécessite d'être connecté) :: POST /tache/ajout


{
  "name" : "test",
  "deadline" : "2069-05-24T12:12:12"
}

Voir les tâches du compte actuel (nécessite d'être connecté) :: GET /tache/accueil

Voir le détail d'une tâche :: GET /tache/detail/{id}

Mettre à jour le % d'avancement :: GET /tache/progres/{id}/{valeur}

id est l'identifiant de la tâche et valeur est la nouvelle valeur d'avancement.
